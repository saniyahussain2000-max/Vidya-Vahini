# Vidya-Vahini Commute Tracker

Vidya-Vahini is an Android application designed to track commutes, featuring a Route Dashboard, Ping Hub, and Route Selection. It is built with modern Android development practices using Kotlin and Jetpack Compose.

## Features

* **Route Dashboard**: Overview of commute routes.
* **Ping Hub**: Communication and status updates for commutes.
* **Route Selection**: Interface to select and manage routes.

## Technologies Used

* Kotlin
* Jetpack Compose
* Android SDK
* Gradle

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Sync the project with Gradle files.
4. Build and run the project on an emulator or physical device.
